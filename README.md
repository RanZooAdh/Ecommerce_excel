# 🐞 E-Commerce Bug Report Documentation

## 📌 Project Overview

This repository contains a collection of **manual test cases and bug reports** identified in a mock e-commerce application (`mock_ecommerce_buggy.html`).
The goal of this project is to demonstrate **software testing skills**, including:

* Test case design
* Bug identification and reporting
* Functional validation of web applications

---

## 🧪 Test Scope

The following modules were tested:

* Shopping Cart
* Promo Code
* Order Summary
* Checkout
* Payment Validation

---

## 🚨 Summary of Issues

All test cases executed resulted in **failures**, indicating multiple critical issues in the application.

### Key problem areas:

* Incorrect pricing and shipping logic
* Invalid promo code acceptance
* Quantity handling issues
* Lack of validation in checkout fields
* Inconsistent pricing between pages
